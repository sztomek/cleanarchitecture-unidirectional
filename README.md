# cleanarchitecture-unidirectional
Sample project to demonstrate an implementation of the CLEAN architecture with unidirectional data flow on Android.

# Used technologies
* MVP architecture
* Unidirectional data flow
* RxJava2 + RxAndroid + RxBinding
* Dagger2
* Retrofit2
* Timber
* Butterknife
* Firebase-auth
* mockito
